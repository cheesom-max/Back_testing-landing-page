# PRD: AI 백테스팅 서비스 랜딩 페이지

## 1. 개요

### 1.1 제품명
AI 백테스팅 랩 (AI Backtesting Lab)

### 1.2 제품 설명
투자자들이 자신의 투자 전략을 과거 10년 데이터로 미리 검증할 수 있는 AI 기반 백테스팅 서비스의 랜딩 페이지

### 1.3 목표
- 서비스 가치 전달 및 사용자 전환 (리드 수집)
- 평균 손실 방지 금액 173만원의 가치 강조
- 3분 만에 완료되는 간편한 사용성 어필

### 1.4 타겟 사용자
| 구분 | 설명 |
|------|------|
| 주요 타겟 | 개인 투자자 (주식, ETF, 코인) |
| 연령대 | 30~50대 |
| 특징 | 투자 경험 있으나 손실 경험 보유, 코딩 지식 없음 |
| 페인 포인트 | 전략에 대한 확신 부족, 감정적 매매, 타이밍 실패 |

## 5. 섹션별 상세 명세

### 5.1 Section 1: Hero (손실 경고)
**목적**: 첫인상 형성, 핵심 가치 전달

| 요소 | 내용 |
|------|------|
| 배경 | 어두운 그라데이션 (gray-900 → gray-800) |
| 헤드라인 | "백테스팅 안 하고 투자하면 잃어버리는 것들" |
| 서브 카피 | 다음 매매 평균 손실액: 267,000원 |
| 비교 박스 | 백테스팅 없음 -27% vs 백테스팅 사용 +18% |
| CTA | "무료로 내 전략 검증하기" (→ Final CTA로 스크롤) |
| 신뢰 요소 | 3분 소요, 카드 불필요, 5,847명 검증 완료 |
| 면책 조항 | 과거가 미래를 100% 보장하지 않음 고지 |

### 5.2 Section 2: Yes Set (마인드 리딩)
**목적**: 공감 유도, "예"라고 답하게 만들기

| 요소 | 내용 |
|------|------|
| 헤드라인 | "혹시 이런 생각, 해보신 적 있으신가요?" |
| 체크리스트 | 10개 항목 (투자 실패 경험 공감) |
| 전환 문구 | "하나라도 체크하셨다면, 백테스팅이 답을 드릴 수 있습니다" |

**체크리스트 항목**:
1. "이번엔 진짜 오를 것 같은데..." → -8% 손실
2. 매수하면 떨어지고, 매도하면 오르는 경험
3. "조금만 더 기다리면..." → -5%에서 -23%로 확대
4. 유튜브/커뮤니티 전략 따라했다 손해
5. "내 전략이 맞긴 한 건가?" 의심
6. 손절 라인 지키지 못함
7. "이번엔 다르다" 물타기 실패
8. 수익 났을 때 너무 일찍 매도
9. "여기서 샀어야 했는데" 후회
10. 왜 손해보는지 이유를 모름

### 5.3 Section 3: Problem Deepening (문제 심화)
**목적**: 문제의 근본 원인 인식

| 요소 | 내용 |
|------|------|
| 헤드라인 | "이 고민들의 공통점이 뭘까요?" |
| 3개 카드 | 확신 부족, 타이밍 실패, 감정 매매 |
| 인사이트 박스 | "이 모든 문제의 원인은 '검증하지 않았기 때문'" |

### 5.4 Section 4: Solution Overview (백테스팅 소개)
**목적**: 솔루션 쉽게 설명

| 요소 | 내용 |
|------|------|
| 헤드라인 | "백테스팅, 쉽게 설명하면" |
| 정의 | 과거 10년 데이터로 전략 테스트 |
| 비유 | "시험 전 모의고사처럼" |
| 3단계 | 전략 입력 → AI 시뮬레이션 → 결과 확인 |

### 5.5 Section 5: Value Proof 1 (손실 방지)
**목적**: 실제 손실 방지 사례 제시

| 요소 | 내용 |
|------|------|
| 아이콘 | 🛡️ |
| 사례 | RSI 30 전략 → 백테스팅 결과 -23% → 287만원 손실 방지 |
| 통계 | 94%가 전략 수정, 평균 173만원 손실 방지 |

### 5.6 Section 6: Value Proof 2 (확신 생성)
**목적**: 데이터 기반 확신의 중요성

| 요소 | 내용 |
|------|------|
| 아이콘 | 💡 |
| 비교 | 백테스팅 전 (실패 반복) vs 후 (전략 유지 → 수익) |
| 통계 | 전략 유지율 87% vs 22%, 수익률 74% vs 23% |

### 5.7 Section 7: Value Proof 3 (최적화)
**목적**: 타이밍 최적화 효과

| 요소 | 내용 |
|------|------|
| 아이콘 | 🎯 |
| Before/After | RSI 30 → RSI 25 = 수익률 2배 |
| 통계 | 수익률 +32%, MDD -47%, 거래 -23% |

### 5.8 Section 8: Value Proof 4 (감정 제거)
**목적**: AI의 객관적 판단 가치

| 요소 | 내용 |
|------|------|
| 아이콘 | 🤖 |
| 대비 | 투자자 기대감 vs AI 냉정한 분석 |
| 결과 | 340만원 손실 방지, 감정적 매매 73% 감소 |

### 5.9 Section 9: How It Works (상세 사용법)
**목적**: 3분 사용법 시각화

**STEP 1: 전략 입력 (30초)**
- 지표 선택 (이동평균선, RSI, MACD)
- 매수 조건 입력
- 손절 라인 설정
- 코딩 지식 0% 필요

**STEP 2: AI 자동 분석 (2분)**
- 프로그레스 바 애니메이션
- 2014~2024 데이터 로딩
- 3,650일 시뮬레이션
- 수수료 자동 계산

**STEP 3: 결과 확인 (즉시)**
- 핵심 지표 (수익률, MDD, 승률)
- 위험 구간 (코로나 등 이벤트)
- AI 개선 제안

### 5.10 Section 10: Objection Buster (반박 제거)
**목적**: 의심/우려 해소

| 질문 | 핵심 답변 |
|------|-----------|
| 코딩 못 하는데? | 클릭 3번, 50대 컴맹도 3분 완료 |
| 과거가 미래 보장? | 100% 아니지만 손실 확률 73% ↓ |
| 비쌀 것 같은데? | 0원 (경쟁사 49,000원) |
| 내 전략은 특수한데? | 50개+ 지표, 커스텀 조건, AI 추천 |
| 결과 신뢰할 수 있어? | 10년 데이터, 97% 정확도 |
| 결과가 나쁘면? | 나쁜 결과 = 좋은 소식 (손실 방지) |
| 지금 당장 필요해? | 3분 vs 26.7만원 손실 위험 |
